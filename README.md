# Generator-hero

A generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-hero`
- Run: `yo hero`
- Make a module: `yo hero:module`
- Make a component: `yo hero:component`
